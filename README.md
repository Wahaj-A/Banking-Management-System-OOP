# Banking Management System (OOP)

## Overview

This project is a Banking Management System developed in C++ as part of an Object-Oriented Programming (OOP) course. It simulates basic banking operations such as account creation, deposits, withdrawals, money transfers, and customer management while demonstrating core OOP principles.

---

## Features

- Create bank accounts
- Customer management
- Deposit money
- Withdraw money
- Transfer funds between accounts
- Savings and Current account support
- Loan management
- Balance inquiry
- Transaction processing

---

## OOP Concepts Implemented

This project demonstrates the practical implementation of the following Object-Oriented Programming concepts:

- Classes and Objects
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Abstract Classes
- Virtual and Pure Virtual Functions
- Function Overloading
- Function Overriding
- Friend Functions and Friend Classes
- Association
- Composition
- Dynamic Memory Allocation
- Pointers
- Static and Constant Members
